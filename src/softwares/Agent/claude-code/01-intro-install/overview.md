# Claude Code 介绍

Claude Code 是 Anthropic 推出的命令行 AI 编程助手。与 ChatGPT 不同，它不是只能聊天——它能直接读写你的文件、执行命令、操作 Git，像一个能动手的副手。

## 能做什么

- 读取数据文件，写脚本处理，运行，画图
- 搜索代码库，定位问题，修改后验证
- 自然语言操作 Git：提交、分支、回退
- 安装第三方 Skill 扩展专业能力
- 并行派发子 Agent 处理独立任务

## 课题组适用场景

- 实验数据处理与可视化（TG、FTIR、XRD 等）
- 批量文件重命名、格式转换
- 文献管理与知识库搭建
- 论文 LaTeX 表格生成、图注润色
- 计算脚本编写与调试

## 接入方式

本教程使用 DeepSeek API 作为后端，无需海外信用卡，人民币充值即可使用。DeepSeek 兼容 Anthropic Messages API 协议，Claude Code 仅需修改环境变量即可接入。
