# 常用命令

## 斜杠命令

| 命令 | 作用 |
|------|------|
| `/init` | 扫描项目，自动生成 `CLAUDE.md` |
| `/plan` | 先出方案，确认后再执行 |
| `@文件/文件夹` | 将文件内容注入对话上下文 |
| `/clear` | 清空对话历史，开启全新会话 |
| `/compact` | 压缩上下文，保留关键决策 |
| `/model` | 切换模型 |
| `/rewind` | 回退到之前的对话节点 |
| `/resume` | 恢复历史会话 |
| `/rename` | 给当前会话命名 |
| `/branch` | 从当前对话创建并行分支 |
| `/goal` | 设定目标条件，达成后自动停止 |
| `/diff` | 交互式查看变更差异 |
| `/cost` | 显示当前会话 token 用量和费用 |
| `/config` | 打开设置（主题、模型、输出风格） |
| `/permissions` | 管理工具权限 |
| `/tasks` | 查看后台任务列表 |
| `/memory` | 编辑 AI 记忆文件 |
| `/plugin` | 安装和管理插件 |
| `/skills` | 列出已安装的 skill |
| `/doctor` | 运行安装健康检查 |
| `/help` | 查看完整帮助 |

## 键盘快捷键

| 快捷键 | 作用 |
|------|------|
| `Esc` / `Ctrl + C` | 停止当前操作 |
| `Tab` | 自动补全文件路径和命令 |
| `Shift + Tab` | 切换 auto-accept / plan 模式 |
| `Ctrl + G` | 打开编辑器编辑 Prompt 或 Plan |
| `!` | 临时执行 shell 命令 |
| `&` | 后台执行命令（`/tasks` 查看进度） |
| `\ + Enter` | 换行 |
