# Claude Code

Claude Code 是命令行 AI 编程助手，能读写文件、执行命令、操作 Git，适合科研数据处理、代码开发和文档编写。

## 使用须知

"推理"可以外包，"理解"不能。AI 在你画的框里跑得很快，但框只有你能画。

- **你画框，AI 跑**：给它明确的验收条件——图要什么风格、数据范围在哪、输出格式是什么。
- **验证留给自己**：不让 AI 替你判断结论的合理性。让它验证自己的代码能不能跑通，你省下时间做只有你能做的判断。
- **方向不外包**：选题、实验设计、论文论证逻辑，这些由你决定。

## 为什么是 Agent

ChatGPT 能回答问题但不能操作文件、执行命令。Claude Code 是 Agent——读完数据能直接写脚本、运行、画图，报错了能查看日志并修正。区别在于工具调用：AI 输出结构化操作指令，串在一起就从"参谋"变成了"副手"。
