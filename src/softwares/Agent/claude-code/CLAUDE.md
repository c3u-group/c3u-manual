# CLAUDE.md

Claude Code 教程，面向课题组（C3U）成员的学术平实风格指南。

## 目录结构

```
intro.md                        # 章节总览
01-intro-install/               # 介绍与安装
02-quick-start/                 # 快速上手
03-advanced-guide/              # 进阶指南（使用者视角——"用"）
04-advanced/                    # 高级用法（开发者视角——"造"）
05-appendix/                    # 附录
```

## 各章边界

- **介绍与安装**：前置知识（终端基本操作）、是什么、适用场景、环境搭建、API 接入、安装验证。
- **快速上手**：首次启动、`/plan` 初体验、常用命令与快捷键、Prompt 技巧。不涉及项目配置和复杂工作流。
- **进阶指南**：使用者视角，安装和配置现成工具。CLAUDE.md 配置、Git 协同、Skill/MCP/Hook 的安装使用与推荐清单，可让 Claude 代写简单版本。只讲"用"，不讲开发。
- **高级用法**：开发者视角，自己构建或指挥 Claude 构建复杂扩展。插件开发（多 Subagent + 多 script 组合体，含模型选择、注意力稀释、指令遵循性）、Hooks 进阶、MCP Server 开发、CI/CD 集成。涉及架构设计的归这里。
- **附录**：纯速查（表格、清单）、FAQ、故障排查、完整示例。论述性内容留给正文章节。

## 写作规范

- 中文正文，代码/命令/路径保留原文
- 学术平实风格，不用 emoji，不堆砌感叹号
- 每个文件独立可读，交叉引用用相对路径
- 读者假设：不要求命令行经验，不要求编程背景
