# 安装 Claude for Office 插件

## 通过 Microsoft AppSource 安装

1. 打开 Word、Excel 或 PowerPoint
2. 点击功能区 **插入** → **获取加载项**（或 **Office 加载项**）
3. 在搜索框中输入 "Claude"
4. 找到 **Claude for Excel, PowerPoint, and Word**（发布者：Anthropic）
5. 点击 **添加**，按提示完成安装

安装完成后，功能区会出现 Claude 面板入口。

## 首次启动

1. 点击功能区 **Claude** 图标，右侧弹出 Claude 面板
2. 出现登录界面，**不要选择** "Sign in with Anthropic"
3. 选择下方的 **LLM Gateway** 选项（稍后在[配置插件](./04-office-config.md)中填入代理地址）
