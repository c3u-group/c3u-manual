# 常用命令

## 斜杠命令

| 命令 | 作用 |
|------|------|
| **任务** | |
| `/init` | 扫描项目，创建 `CLAUDE.md` |
| `/plan` | 先出方案，确认后再执行 |
| `@文件/文件夹` | 指定要操作的文件或文件夹（前后加个空格） |
| `/goal` | 设定目标条件，Haiku 逐轮评估，达成后自动停止 |
| **会话控制** | |
| `/clear` | 在不相关的任务之间，清空上下文，开启全新会话 |
| `/model` | 切换模型，简单任务用 Haiku 更快更省 |
| `/rewind` 或 两次 `Esc` | 回退到之前的对话节点 |
| **分支协作** | |
| `/branch` | 创建分支，进行不同修改，后续合并 |
| `/btw` | 临时询问，可切换分支 |
| **管理** | |
| `/context` | 显示上下文用量明细 |
| `/permissions` | 管理工具权限，预设允许/拒绝 |
| `/tasks` | 查看后台任务列表 |
| `/memory` | 管理 AI 记住的偏好和规范 |
| `/plugin` | 安装和管理插件 |
| `/skills` | 列出安装的 skill |
| `/help` | 查看完整帮助 |

## 键盘快捷键

| 快捷键 | 作用 |
|------|------|
| `Esc` 或 `Ctrl + C` | 停止当前操作 |
| `Tab` | 自动补全 |
| `Shift + Tab` | `auto-accept` / `plan` 模式切换 |
| `Ctrl + G` | 打开编辑器（编辑 Prompt / Plan） |
| `Ctrl + 点击` | 打开链接 |
| `!` | 临时执行命令 |
| `&` | 后台执行命令（用 `/tasks` 查看进度） |
| `\ + Enter` | 换行 |
