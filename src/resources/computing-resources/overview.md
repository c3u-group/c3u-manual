# 计算资源

## 服务器概览

课题组共有 8 台服务器：

| 编号 | 系统 |
|------|------|
| 1-6 号 | Windows |
| 7 号 | Linux (RHEL 9) |
| 8 号 | Linux (Rocky Linux 9) |

7、8 号机为 Linux 系统，部署在实验室机柜中，主要通过校园网 SSH 远程访问，承担课题组主要计算任务。以下介绍均基于 Linux 系统机器。

## 账号申请

每人分配独立普通账号，新用户联系管理员创建。如需安装需要超级用户权限的软件，联系管理员或参考 [软件环境](./software.md) 中的 Apptainer 方案。
